# 公众号文章数据助手

一个纯 Edge / Chrome 插件，用于导出微信公众号文章运营简表。

它只做一件事：按日期范围拉取公众号文章数据，并导出 CSV 到浏览器默认下载目录。

## 功能

- 选择开始日期和结束日期。
- 使用微信官方公众号数据接口拉取文章数据。
- 导出文章标题、日期、阅读数、点赞、转发、喜欢、留言等字段。
- 支持字段勾选。
- 支持自定义列顺序。
- 支持两种口径：
  - 每篇文章一行，取最后统计日累计值。
  - 保留每日累计明细。
- 导出 CSV，可直接用 Excel 打开。

## 不包含

- 地域数据批量下载。
- 地域统计战报。
- 视频号截图统计。
- 本地 Python 服务。

## 安装方式

1. 下载本仓库。
2. 解压后打开 Edge 扩展管理页：

```text
edge://extensions/
```

3. 打开“开发人员模式”。
4. 点击“加载解压缩的扩展”。
5. 选择本项目里的 `share-extension` 文件夹。

Chrome 用户也可以在：

```text
chrome://extensions/
```

按相同方式加载。

## 使用前准备

1. 确认公众号是已认证账号，并拥有对应数据接口权限。
2. 在微信公众平台进入“设置与开发 / 基本配置”，获取 AppID 和 AppSecret。
3. 把当前电脑的出口 IP 加入公众号后台的 IP 白名单。
4. 点击插件图标，进入“工具设置”，填写 AppID 和 AppSecret。

## 使用方式

1. 点击浏览器右上角插件图标。
2. 选择日期范围。
3. 点击“获取公众号数据”。
4. CSV 会下载到浏览器默认下载目录。

默认文件名：

```text
wechat_articles_selected_开始日期_结束日期.csv
```

## 字段和列顺序

点击插件里的“工具设置”，可以设置：

- 数据接口。
- 运营简表口径。
- 导出字段。
- 列顺序。

设置保存在当前浏览器的插件存储中。

## 隐私和安全

- AppID 和 AppSecret 只保存在当前浏览器的插件存储中。
- 本插件不会上传 AppID、AppSecret 或文章数据到第三方服务器。
- 插件会直接请求微信官方接口：`https://api.weixin.qq.com/*`。
- 如果你把插件分享给别人，对方需要填写自己的 AppID 和 AppSecret。

## 常见问题

### 提示 40164

当前电脑的出口 IP 不在公众号 IP 白名单中。

请到微信公众平台：

```text
设置与开发 / 基本配置 / IP 白名单
```

添加当前网络出口 IP。

### 提示 AppSecret 错误

请检查工具设置里的 AppID 和 AppSecret 是否属于同一个公众号。

### 导出为空

可能原因：

- 所选日期没有发表文章。
- 当天数据尚未完成统计。
- 结束日期不能超过昨天。

微信数据通常建议在上午 8 点后查询前一天数据。
