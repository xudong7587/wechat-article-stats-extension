# 公众号文章数据助手

一个纯 Edge / Chrome 插件，用于导出微信公众号文章运营简表，并支持把视频号“动态数据明细 CSV”合并进同一张表。

当前版本：`1.1.1`

## 功能

- 按日期范围拉取公众号文章数据。
- 导出文章标题、日期、阅读/播放数、点赞/推荐、转发、喜欢、留言等字段。
- 支持字段勾选和自定义列顺序。
- 支持两种公众号口径：
  - 每篇文章一行，取最后统计日累计值。
  - 保留每日累计明细。
- 支持视频号统计：
  - 先从视频号后台下载“视频号动态数据明细.csv”。
  - 在插件中选择该 CSV。
  - 可导出“仅视频号数据”，也可与公众号数据“一键统计全部数据”合并导出。
- 导出 CSV，可直接用 Excel 打开。
- 数字字段会在导出前统一清洗，减少 Excel 打开后被识别成文本的情况。

## 安装方式

1. 下载本仓库。
2. 解压后打开 Edge 扩展管理页：

```text
edge://extensions/
```

3. 打开“开发人员模式”。
4. 点击“加载解压缩的扩展”。
5. 选择本项目里的 `share-extension` 文件夹。

Chrome 用户也可以在：

```text
chrome://extensions/
```

按相同方式加载。

## 使用前准备

1. 确认公众号是已认证账号，并拥有对应数据接口权限。
2. 在微信公众平台进入“设置与开发 / 基本配置”，获取 AppID 和 AppSecret。
3. 把当前电脑的出口 IP 加入公众号后台的 IP 白名单。
4. 点击插件图标，进入“工具设置”，填写 AppID 和 AppSecret。

## 导出公众号数据

1. 点击浏览器右上角插件图标。
2. 选择开始日期和结束日期。
3. 点击“仅公众号数据”。
4. CSV 会下载到浏览器默认下载目录。

默认文件名：

```text
wechat_articles_selected_开始日期_结束日期.csv
```

## 导出视频号数据

视频号数据不是通过公众号开发接口获取，而是通过视频号后台导出的 CSV 文件获取。

使用步骤：

1. 打开视频号后台的数据页面。
2. 进入单篇数据 / 动态数据明细相关页面。
3. 在视频号页面中先选择好日期范围。
4. 下载“视频号动态数据明细.csv”。
5. 回到插件，在“视频号 CSV”里选择刚下载的 CSV。
6. 点击“仅视频号数据”。

默认文件名：

```text
wechat_channels_selected_开始日期_结束日期.csv
```

## 一键统计全部数据

1. 先按“导出视频号数据”的步骤，在视频号页面设置日期范围并下载 CSV。
2. 在插件里选择同一个 CSV。
3. 在插件中选择公众号统计日期范围。
4. 点击“一键统计全部数据”。

插件会：

- 读取视频号 CSV。
- 调用微信官方公众号接口拉取公众号文章数据。
- 按插件中的日期范围过滤视频号 CSV。
- 合并导出一张 CSV。

默认文件名：

```text
wechat_articles_channels_selected_开始日期_结束日期.csv
```

## 视频号字段映射

视频号 CSV 会按以下规则合并到运营简表：

- 视频描述 -> 文章标题
- 发布时间 -> 文章日期
- 播放量 -> 阅读/播放数
- 推荐 -> 点赞/推荐
- 分享量 -> 转发
- 喜欢 -> 喜欢
- 评论量 -> 留言/评论

注意：视频号里的“大拇指”是“推荐”，“爱心”是“喜欢”，和公众号页面上的展示顺序不完全一样。

## 字段和列顺序

点击插件里的“工具设置”，可以设置：

- 数据接口。
- 运营简表口径。
- 导出字段。
- 列顺序。

设置保存在当前浏览器的插件存储中。

## 隐私和安全

- AppID 和 AppSecret 只保存在当前浏览器的插件存储中。
- 本插件不会上传 AppID、AppSecret 或文章数据到第三方服务器。
- 插件会直接请求微信官方接口：`https://api.weixin.qq.com/*`。
- 视频号 CSV 只在本地浏览器中读取和处理。
- 如果你把插件分享给别人，对方需要填写自己的 AppID 和 AppSecret。

## 常见问题

### 提示 40164

当前电脑的出口 IP 不在公众号 IP 白名单中。

请到微信公众平台：

```text
设置与开发 / 基本配置 / IP 白名单
```

添加当前网络出口 IP。

### 提示 AppSecret 错误

请检查工具设置里的 AppID 和 AppSecret 是否属于同一个公众号。

### 公众号导出为空

可能原因：

- 所选日期没有发表文章。
- 当天数据尚未完成统计。
- 结束日期不能超过昨天。

微信数据通常建议在上午 8 点后查询前一天数据。

### 视频号数据缺少日期

视频号 CSV 的日期范围由视频号后台页面当前选择决定。  
如果页面默认是 7 天，CSV 通常只包含 7 天。需要更长范围时，请先在视频号页面选择对应日期范围，再下载 CSV。
