# 公众号文章数据助手分享版

版本：1.1.2

这是一个纯 Edge / Chrome 插件版工具，不依赖 Python 本地服务。

## 功能

- 导出微信公众号文章运营简表。
- 支持自定义字段和列顺序。
- 支持从视频号“数据中心 - 单篇数据”页面自动读取动态数据明细。
- 支持公众号和视频号数据合并导出。

## 不包含

- 不包含地域战报功能。
- 不包含本地 Python 服务。
- 不包含个人 AppID、AppSecret、缓存、日志或导出数据。

## 安装

1. 打开 Edge 扩展管理页：

```text
edge://extensions/
```

2. 开启“开发人员模式”。
3. 点击“加载解压缩的扩展”。
4. 选择本文件夹下的：

```text
edge-extension
```

## 首次设置

1. 点击插件图标。
2. 点击“工具设置”。
3. 填写使用者自己的公众号 AppID 和 AppSecret。
4. 选择数据接口、统计口径、字段和列顺序。
5. 点击“保存设置”。

密钥只保存在当前浏览器的插件存储里，不会写入插件文件。

## 使用

### 公众号数据

在插件中选择日期范围，点击“仅公众号数据”。

### 视频号数据

1. 打开视频号数据中心的单篇数据页面：

```text
https://channels.weixin.qq.com/platform/statistic/post
```

2. 在视频号页面选择日期范围。
3. 回到插件，点击“仅视频号数据”或“一键统计全部数据”。

插件会尝试在视频号页面自动触发导出，并直接读取页面生成的明细数据，不需要手动选择 CSV 文件。

## 注意

- 微信接口报 `40164` 时，需要把当前出口 IP 添加到公众号后台 IP 白名单。
- 视频号页面的日期范围需要先在视频号后台页面设置好；插件里的日期范围用于筛选最终导出结果。
- 如果微信页面调整了导出按钮或浏览器阻止页面内读取导出文件，插件会提示无法读取视频号明细。
- 本分享版不提供地域战报。
