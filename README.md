# 公众号文章数据助手

版本：1.3.0

这是一个 Edge / Chrome 浏览器插件，用来导出微信公众号文章数据，并可合并视频号单篇数据。本分享版只保留公众号和视频号统计导出，不包含本地服务、地域战报和飞书同步。

## 功能

- 导出微信公众号文章阅读、点赞、转发、喜欢、留言等运营数据。
- 通过发布列表接口补充发布时间、发布位置、是否头条、是否已删除等元数据；可识别到已删除的公众号文章会自动剔除。
- 支持自定义导出字段和列顺序。
- 支持从视频号“数据中心 - 单篇数据”页面自动读取动态数据明细。
- 支持公众号和视频号数据合并导出。
- 导出的表格会保存到浏览器默认下载目录。

## 安装

1. 打开 Edge 扩展管理页：

```text
edge://extensions/
```

2. 开启“开发人员模式”。
3. 点击“加载解压缩的扩展”。
4. 选择本文件夹下的：

```text
edge-extension
```

## 首次设置

1. 点击插件图标。
2. 点击“工具设置”。
3. 填写你自己的公众号 AppID 和 AppSecret。
4. 选择数据接口、统计口径、字段和列顺序。
5. 点击“保存设置”。

AppID 和 AppSecret 只保存在当前浏览器的插件存储里。

## 使用

### 公众号数据

在插件中选择日期范围，点击“仅公众号数据”。

### 视频号数据

1. 打开视频号数据中心的单篇数据页面：

```text
https://channels.weixin.qq.com/platform/statistic/post
```

2. 在视频号页面选择日期范围。
3. 回到插件，点击“仅视频号数据”或“一键统计全部数据”。

插件会尝试在视频号页面自动触发导出，并直接读取页面生成的明细数据，不需要手动选择 CSV 文件。

## 注意事项

- 微信接口报 `40164` 时，需要把当前出口 IP 添加到公众号后台 IP 白名单。
- 视频号页面的日期范围需要先在视频号后台页面设置好；插件里的日期范围用于筛选最终导出结果。
- 如果微信页面调整了导出按钮或浏览器阻止页面内读取导出文件，插件会提示无法读取视频号明细。
