# 公众号数据助手分享版

这是一个 Windows + Edge/Chrome 使用的微信公众号、视频号运营数据导出工具。

当前分享版由自用版 `0.4.4` 剥离，不包含个人密钥、缓存、日志或导出数据。

## 功能

- 获取公众号文章阅读、点赞、转发、喜欢、留言等数据。
- 支持自定义导出字段和列顺序。
- 支持自动下载视频号单篇数据 CSV，并合并公众号与视频号数据。
- 支持下载公众号后台地域 Excel，并生成地域战报。
- 视频号 CSV 和地域下载临时文件在处理完成后自动清理。

## 安装

1. 安装 Python 3.10 或更高版本。
2. 在本文件夹打开 PowerShell。
3. 安装本地伴随服务：

```powershell
.\install_local_service.ps1
```

4. 打开 Edge 扩展管理页：

```text
edge://extensions/
```

5. 打开“开发人员模式”，选择“加载解压缩的扩展”，选择：

```text
edge-extension
```

## 首次设置

1. 点击浏览器插件图标。
2. 点击“工具设置”。
3. 填写公众号 `AppID` 和 `AppSecret`。
4. 设置默认导出路径。
5. 按需要勾选字段并调整列顺序。
6. 保存设置。

密钥会写入本机 `.env`，不会进入插件代码。

## 使用

### 公众号数据

在插件中选择日期范围，点击“仅公众号数据”。

### 视频号数据

1. 打开视频号数据页面：

```text
https://channels.weixin.qq.com/platform/statistic/post
```

2. 在页面里选择好日期范围。
3. 回到插件点击“仅视频号数据”或“一键统计全部数据”。

### 地域战报

1. 打开公众号后台页面，确保插件能识别当前后台 token。
2. 在插件地域战报区域设置日期和地域 Excel 文件夹。
3. 点击“下载并生成战报”。

## 注意

- 微信接口报 `40164` 时，需要把当前出口 IP 添加到公众号后台 IP 白名单。
- 视频号数据来自页面下载的 CSV，请先在视频号页面设置好日期范围。
- 地域数据来自公众号后台下载的地域 Excel。
- 不要把包含 `.env` 的个人备份分享给别人。
